# NeuroMatrix Mini App

Небольшое React/Vite-приложение для Telegram Mini App: каталог нейросетей по задачам с быстрыми ссылками.

## Быстрый старт
1) Установите зависимости:
```
npm install
```
2) Запустите dev-сервер:
```
npm run dev
```
3) Откройте адрес из терминала (обычно http://localhost:5173). В Telegram Bot API укажите этот URL в качестве `WebApp` или задеплойте на хостинг и используйте внешнюю ссылку.

## Что внутри
- Поиск по названию, описанию и тегам.
- Фильтр по категории.
- Карточки с описанием и ссылкой (открываются в новом окне).
- Автоподстройка цветов под `Telegram.WebApp.themeParams`, если приложение открыто в Telegram.

## Структура
- `src/data/networks.ts` — список сервисов (название, описание, категория, ссылка, теги).
- `src/components/NetworkCard.tsx` — карточка элемента.
- `src/App.tsx` — фильтры + сетка карточек.
- `src/styles.css` — стилизация, тёмная тема по умолчанию.

## Настройка под себя
- Добавьте или измените сервисы в `src/data/networks.ts`.
- При необходимости замените цвета в `src/styles.css`; если Mini App открыт внутри Telegram, будет использоваться тема клиента.

## Проверка
- `npm run build` — сборка.
- `npm run preview` — предпросмотр собранной версии.
