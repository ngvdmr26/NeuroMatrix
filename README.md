# NeuroMatrix Mini App

React/Vite mini app для Telegram WebApp: каталог нейросетей по задачам с быстрыми ссылками и современным UI.

## Быстрый старт
1) Установите зависимости:
```
npm install
```
2) Запустите dev-сервер:
```
npm run dev
```
3) Откройте адрес из терминала (обычно http://localhost:5173). В Telegram Bot API укажите этот URL в качестве `WebApp` или задеплойте на хостинг и используйте внешнюю ссылку.

## Что внутри
- Поиск по названию, описанию и тегам.
- Фильтры по категориям (чипы).
- Карточки с логотипами, описанием и ссылкой (открываются в новом окне).
- Вступительная анимация-интро про цели проекта.
- Автоподстройка цветов под `Telegram.WebApp.themeParams`, если приложение открыто в Telegram.

## Структура
- `src/data/networks.ts` — список сервисов (название, описание, категория, ссылка, логотип, теги).
- `src/components/NetworkCard.tsx` — карточка с логотипом и тегами.
- `src/App.tsx` — интро-анимация, фильтры + сетка карточек.
- `src/styles.css` — стилизация, современный тёмный/градиентный стиль.

## Настройка под себя
- Добавьте или измените сервисы в `src/data/networks.ts`.
- При необходимости замените цвета в `src/styles.css`; если Mini App открыт внутри Telegram, будет использоваться тема клиента.

## Проверка
- `npm run build` — сборка.
- `npm run preview` — предпросмотр собранной версии.
